## 2. 필수 구현 조건 (Conditions)

프로젝트는 아래 **3가지 기술 요소를 모두** 포함해야 한다.

### 2.1. CI/CD 자동 배포 환경
- **GitHub Actions** 등을 활용한 자동 배포 파이프라인 구성
- `main` 브랜치 푸시 시 빌드 → 테스트 → 배포까지 자동 실행

### 2.2. 인프라/배포 기술
- **Docker**: 컨테이너화 (Dockerfile 작성)
- **Nginx**: 리버스 프록시, 정적 파일 서빙 등
- 수업에서 학습한 기술 요소를 **적극 활용**

### 2.3. 데이터 저장
- **Web Storage 활용 (필수)**: 아래 중 1개 이상
  - `localStorage`
  - `sessionStorage`
  - `IndexedDB`
- **DBMS 활용 (필수)**: 서버 측 데이터베이스 사용
  - 예: PostgreSQL, MySQL, MongoDB 등
